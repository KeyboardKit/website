KeyboardKit helps you build amazing keyboards that look and behave just like the native iOS keyboards...or completely different. It extends the native APIs, adds a bunch of extra features and can be used on all major Apple platforms.

KeyboardKit is open-source and completely free, but you can upgrade to KeyboardKit Pro to unlock pro features like fully localized keyboards, local and remote autocomplete, additional views, AI supporting features etc.
