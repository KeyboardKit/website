KeyboardKit is a Swift-based SDK that helps you build custom keyboard extensions for iOS with Swift and SwiftUI. It extends the native keyboard APIs and provides you with a lot of functionality that can be used on all major Apple platforms, like iOS, macOS, tvOS and watchOS.

Features

KeyboardKit provides you with native-like keyboards and a bunch of tools and services.

Locales

KeyboardKit supports {% include locales/count.html %} locales, including left-to-right and right-to-left ones.

Open-Source

KeyboardKit is open-source, completely free and hosted at the amazing GitHub.

KeyboardKit Pro

KeyboardKit Pro unlocks pro features like fully localized keyboards, autocomplete etc.

App

The KeyboardKit app let you create and style keyboards directly on your iPhone and iPad.
