KeyboardKit is a Swift-based SDK that helps you build custom keyboard extensions for iOS with Swift and SwiftUI. It extends the native keyboard APIs and provides you with a lot of functionality that can be used on all major Apple platforms, like iOS, macOS, tvOS and watchOS.

Open-Source

KeyboardKit is open-source and completely free. It's hosted at GitHub, where you find source code, documentation, a demo app etc.

Features

KeyboardKit has {% include locales/count.html %} locales, system keyboards, emojis, input sets, keyboard layouts, gestures, haptics, autocomplete, extensions etc.

KeyboardKit Pro

KeyboardKit Pro extends KeyboardKit with pro features, like localized system keyboards, local and remote autocomplete, proxy utils etc.

App

The KeyboardKit app let you create keyboards directly on your iPhone and iPad. Download it from the App Store and give it a try.
